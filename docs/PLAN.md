# Callex MVP Plan

## Goal
Build an AI system that can:
- Understand a booking request
- Check available slots
- Create an appointment
- Return a confirmation

This is the first sellable version of Callex.

## MVP Scope
Included:
- Chat-based booking
- API for booking
- Database for appointments
- Simple web UI
- Cloud deployment

Not included:
- Voice
- WhatsApp
- Mobile apps
- Advanced analytics
